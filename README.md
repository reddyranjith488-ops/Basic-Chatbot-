# 🤖 Basic Chatbot - CodeAlpha Internship

A simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.

---

## 📌 Project Description

This project is a console-based chatbot that responds to predefined user inputs such as greetings and simple questions.

The chatbot continues chatting until the user types "bye".

---

## 🚀 Features

- User-friendly chatbot interaction
- Predefined responses
- Continuous conversation loop
- Beginner-friendly Python project
